# guileless-ui

Yet another vue ui components library

## Install the package

```
npm i guileless-ui
```

## Intall tailwindcss

You should install tailwindcss because all the components depend on it.

Refer to [Install TailwindCSS](https://tailwindcss.com/docs/installation)
